# 빚투 모니터

코스피 위험 신호 3가지를 매일 자동으로 체크해서 웹 대시보드로 보여주는 무료 서비스입니다.

**감시 대상 신호:**
1. 일일 반대매매 금액이 200억 → 400억 → 600억으로 상승 추세 진입 여부
2. 신용잔고 증감률이 코스피 수익률을 0.5%p 이상 추월하는 달
3. 삼성전자·SK하이닉스 동시 -3% 일이 5영업일 중 2회 이상 발생

**작동 방식:**
- GitHub의 무료 자동화 기능(Actions)이 매일 평일 한국시간 17:30에 실행됨
- 한국거래소(pykrx)와 네이버 금융에서 데이터를 자동 수집
- 결과를 웹페이지(GitHub Pages)에 띄움 → 휴대폰/PC 어디서든 접속 가능
- 비용 0원, 서버 0대 필요 없음

## 처음 설정하기 (한 번만 하면 됩니다)

### 1단계: GitHub 계정 만들기

이미 있으면 다음 단계로.

1. https://github.com/signup 에서 가입 (이메일·비밀번호·아이디만 있으면 됨)
2. 무료 플랜 선택

### 2단계: 새 저장소(Repository) 만들기

1. GitHub 로그인 → 오른쪽 위 `+` 아이콘 → `New repository` 클릭
2. 다음과 같이 입력:
   - **Repository name**: `bittu-monitor` (마음대로 정해도 됨)
   - **Public** 선택 (Private이면 GitHub Pages 무료 제공 안 됨)
   - **Add a README file** 체크는 **하지 마세요** (덮어쓸 거라서)
3. `Create repository` 클릭

### 3단계: 파일 업로드

이 폴더 안의 모든 파일을 새 저장소에 올립니다.

**방법 A — 웹에서 직접 (가장 쉬움):**
1. 방금 만든 저장소 페이지에서 `uploading an existing file` 링크 클릭
   - (없으면 `Add file` → `Upload files`)
2. 이 폴더의 모든 파일·폴더를 **드래그**해서 올리기
   - `monitor.py`, `requirements.txt`, `.gitignore`, `README.md`
   - `.github/` 폴더 (안에 workflows/daily.yml)
   - `data/` 폴더 (안에 history.csv)
   - `docs/` 폴더 (안에 index.html)
3. 아래쪽 `Commit changes` 버튼 클릭

**방법 B — 익숙하면 git 명령으로:**
```bash
git clone https://github.com/본인아이디/bittu-monitor.git
cp -r 다운받은폴더/* bittu-monitor/
cd bittu-monitor
git add .
git commit -m "initial commit"
git push
```

### 4단계: GitHub Actions 권한 켜기

자동 실행이 결과를 저장소에 다시 쓸 수 있도록 권한을 줍니다.

1. 저장소 페이지 위쪽 메뉴에서 `Settings` 클릭
2. 왼쪽 메뉴에서 `Actions` → `General` 클릭
3. 페이지 아래쪽 `Workflow permissions` 항목에서:
   - **Read and write permissions** 선택
   - `Save` 클릭

### 5단계: GitHub Pages 켜기 (대시보드 웹주소 활성화)

1. `Settings` → 왼쪽 메뉴에서 `Pages` 클릭
2. `Source` 항목에서:
   - **Branch**: `main` 선택
   - **Folder**: `/docs` 선택
   - `Save` 클릭
3. 1-2분 기다린 후 페이지 위쪽에 `Your site is live at https://본인아이디.github.io/bittu-monitor/` 같은 주소가 뜸 → 이게 본인 대시보드 주소

### 6단계: 첫 실행 (수동)

스케줄을 안 기다리고 바로 한 번 돌려서 데이터를 채워봅니다.

1. 저장소 위쪽 메뉴에서 `Actions` 클릭
2. 왼쪽에서 `Daily Bittu Monitor` 클릭
3. 오른쪽에 `Run workflow` 드롭다운 → 녹색 `Run workflow` 버튼 클릭
4. 1-2분 기다리면 실행 완료
5. 5단계에서 받은 주소로 접속하면 대시보드가 보입니다

## 자주 묻는 질문

**Q. 정말 무료인가요?**
A. 네. GitHub Actions는 Public 저장소에 한해 월 2,000분 무료(매일 1-2분 사용). GitHub Pages도 Public 저장소는 무료.

**Q. 데이터가 안 들어오면?**
A. Actions 탭에서 실패한 실행을 클릭하면 빨간 ❌ 표시와 함께 로그가 보입니다. 보통 네이버 금융 페이지 구조가 바뀌면 발생하니 그때는 다시 알려주세요.

**Q. 알림(텔레그램/이메일)도 받고 싶어요.**
A. 가능합니다. monitor.py에 텔레그램 봇 호출 코드 추가하면 됩니다 — 필요하시면 추가 요청 주세요.

**Q. 종목을 바꾸고 싶어요 (예: 삼성전자 → 다른 종목).**
A. monitor.py 파일에서 `005930` (삼성전자), `000660` (SK하이닉스) 부분을 원하는 종목 코드로 바꾸시면 됩니다.

**Q. 매일 17:30 말고 다른 시간에 돌리고 싶어요.**
A. `.github/workflows/daily.yml` 파일에서 `cron: '30 8 * * 1-5'` 부분을 수정. 형식은 `분 시 일 월 요일`이고 시간은 UTC 기준 (한국시간에서 9 빼기). 예: 한국 18:00 = UTC 09:00 = `'0 9 * * 1-5'`
